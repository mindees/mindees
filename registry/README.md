# mindees/registry — Community Module Registry (placeholder)

This directory will host the **community module registry**: third-party
capability modules that implement a **stable, versioned, sandboxed ABI**
(WebAssembly Component Model) so that a third-party module *cannot* break a
consumer's build on an OS/runtime bump, and runs capability-scoped.

**Status: planned.** Nothing here is functional yet. The ABI contract and a
sample validated module are scaffolded in later phases (see
[`ROADMAP.md`](../ROADMAP.md), Phases 8/11/12).

This placeholder exists so the repository structure matches the architecture
from day one and the directory has a clear, honest purpose.
